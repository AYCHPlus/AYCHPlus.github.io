Example: Microformats2, forms


<div class="actions">
  <form class="move backlog" action="/items/backlog" method="POST">
    <input name="id" type="hidden" value="4">
    <input title="submit" type="submit" value="Move to backlog">
  </form>
  <form class="move verify next" action="/items/verify" method="POST">
    <input name="id" type="hidden" value="4">
    <input title="submit" type="submit" value="Move to verify">
  </form>
</div>

Workflow definition


            class Item
              def next_states
                case self.status
                when 'backlog'
                  ['working']
                when 'working'
                  ['backlog', 'verify']
                when 'verify'
                  ['working', 'done']
                when 'done'
                  ['working']
                end
              end
            end
